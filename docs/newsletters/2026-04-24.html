トップニュース: DeepSeek V4

何が起きたのか

DeepSeekが DeepSeek-V4 Pro と DeepSeek-V4 Flash を発表しました。これはV3以来初の大規模なアーキテクチャ刷新であり、初めて明確な二層ラインナップを導入したものです。特徴として 100万トークンのコンテキスト、推論モードと非推論モードのハイブリッド、MITライセンス、そして複数の研究者から「今年最も重要、または最も優れたモデル論文」と評された詳細な技術レポートが挙げられます。反応の中での事実的な共通認識は、V4がオープンウェイトの長コンテキスト処理とエージェント的なコーディング性能を大幅に向上させた一方で、全体的にはトップのクローズドモデルにはやや劣るというものです。独立したベンチマーク評価では、V4 Proはオープンウェイトの中で第2位程度とされ、ベンチマークやモードによっては Kimi K2.6 / GLM-5.1 / 強いClaude Sonnetクラス〜Opus相当 に近く、特に長コンテキストとエージェント性能が強いとされています。GPT-5.xやOpus 4.7にどれほど近いか、またこれが「民主化」なのか、それとも複雑すぎて再現が難しいアーキテクチャなのかについては意見が分かれています。

主な事実と技術詳細

議論で繰り返し言及された具体的な技術的主張：

  • 2つのモデル
    • V4 Pro: 総パラメータ数1.6兆 / 有効49B
    • V4 Flash: 総パラメータ数284B / 有効13B
  • コンテキスト
    • 100万トークン（V3.2の128Kから大幅増）
    • 複数の投稿者がこれを最大の成果と位置付け
  • 学習規模
    • トークン数32兆〜33兆
    • パラメータ1.6兆に対して約20トークン/パラメータ
    • 推定プリトレーニング計算量は約1e25 FLOPs
  • 推論モード
    • 3種類の推論モードを提供
    • 「思考/非思考」ハイブリッド構成
  • 長コンテキストアーキテクチャ
    • 新しいハイブリッドアテンションシステム：
    • 共有KVベクトル
    • 圧縮KVストリーム
    • 圧縮トークンへのスパースアテンション
    • 近傍コンテキストへのローカル/スライディングウィンドウアテンション
    • 2倍のKV削減、4倍〜128倍の圧縮、トップkスパースアテンション、128トークンスライディングウィンドウ
    • 100万コンテキストKVキャッシュは9.62GiB（bf16）、V3.2の83.9GiBより8.7倍小さい
    • FP4インデックスキャッシュ＋FP8アテンションキャッシュでさらに約2倍削減
    • 「KVキャッシュが10倍小さい」と要約する声も
  • 量子化 / チェックポイント形式
    • 混合FP4＋FP8
    • MoEエキスパート重みはFP4
    • アテンション/ノーム/ルーターはFP8
    • モデル全体が8×B200ノードに収まる
  • 推論ハードウェア / サービング
    • Blackwell Ultra上で、V4 Proはエージェントワークフローで150TPS以上のユーザーインタラクションを実現
    • vLLMによるDay-0性能パレート公開
    • H200、MI355、B200、B300、GB200/300でのベンチマーク
    • 256GB MacでDeepSeek4-Flash動作
    • MLX量子化版公開
    • 小容量RAM Mac対応への関心は高いが未対応部分あり
    • 多くのローカルスタックがテンソル並列非対応で、V4クラスモデルは推論インフラに負荷大
  • ライセンス / 提供 / 価格
    • MITライセンス
    • APIおよびサードパーティによる迅速な提供
    • V4 Pro価格: 入力100万トークンあたり$1.74 / 出力$3.48
    • V4 Flash価格: 入力$0.14 / 出力$0.28
    • キャッシュヒット価格も提示
    • Huawei Ascend 950スーパーノードがH2に大規模展開されればPro価格は大幅低下の見込み

独立評価とV4の位置付け

独立ベンチマークの総合評価：

  • V4 Pro Max: Artificial Analysis Intelligence Indexで52（V3.2の42から10ポイント増）
    • オープンウェイト推論モデルで第2位（Kimi K2.6の54に次ぐ）
  • V4 Flash Max: 47（Claude Sonnet 4.6相当）
  • GDPval-AA（エージェント的実務）: V4 Proが1554でオープンウェイトトップ
  • AA-Omniscience: V4 Proは-10（V3.2から11ポイント改善）も幻覚率94%、Flashは96%
  • AA Index実行コスト: V4 Pro $1,071、Flash $113
  • AA Indexでの出力トークン使用量: Pro 1.9億、Flash 2.4億（大量トークン使用がコスト面での注意点）

その他評価視点： * Text Arenaで総合第2位、医療・ヘルスケアで1位 * ProはFlashより約30位高いが12倍高価 * SimpleBenchではOpus 4.5相当との見方 * GLM-5.1より優れるがOpus 4.7やGPT-5.4には及ばないとの評価 * Flash@maxはPro@highと推論性能がほぼ同等 * 長時間実行でDeepSeekとKimiが改善 * 一部は否定的な初期印象

事実と意見

事実

  • V4 Pro/FlashはMITライセンス、100万コンテキスト、詳細な技術文書付きで公開
  • 新しい長コンテキストアテンションシステムでKVキャッシュを大幅削減
  • 独立ベンチマークではオープンウェイト上位だがプロプライエタリモデルには劣る
  • 一部評価では大量トークン使用
  • FP4/FP8混合精度で8×B200ノードに収まる
  • vLLMなどによるDay-0サポート

意見

  • フロンティアから4〜5ヶ月遅れとの見方
  • 「トップ3オープン」か「唯一フロンティアに近いオープン」かで議論
  • 「最強のプリトレーニングモデル」との評価は意見
  • 「今年最も重要なAI論文」との声は熱意
  • 「民主化技術ではない」との見方は政治的解釈

論点

1) フロンティアに近いか明確に遅れているか
2) 真の貢献はモデル品質か長コンテキスト設計か
3) 民主化か再現困難か
4) Flashは過小評価されているか

中国、チップ、Huawei、主権の文脈

V4は米中間の計算資源・主権論争の証拠としても扱われました。中国ラボは「離陸」段階に入りつつも5ヶ月以上遅れ、チップ禁輸がギャップ拡大要因との見方。Huawei Ascend 950スーパーノード展開で価格低下予測。NVIDIAのBlackwell戦略との整合性を指摘する声も。

蒸留、学習データ、品質

V4はアーキテクチャ革新か蒸留かで議論。中国ラボのデータ品質へのこだわりがモデルの文章品質に反映。プリトレーニングデータ詳細は限定的。

アーキテクチャ系譜と先行事例

V4は過去の手法（ハッシュルーティング、Mixtral型MoE）を発展させたもの。MoEの有効性を肯定する声も。

技術レポートの重要性

モデルだけでなくレポート品質への賞賛多数。詳細な技術開示はオープンリリースの期待値を引き上げた。

実用上の制約

閉じたフロンティアモデルには総合力で劣る、推論RLは未成熟、サービングは困難、トークン使用量多、API制御制限、複雑すぎて複製困難。

広範な意味

  1. オープンウェイト長コンテキストは実用的に
  2. 中国トップラボはオープンモデルで競争力維持
  3. 「オープン」の基準はチェックポイント公開からフルスタック共同設計へ
