AI Twitterまとめ

Anthropicが大規模な資金調達を発表し、同時にClaude Opus 4.8をリリースしました。

  • 資金面では、AnthropicはシリーズHで650億ドルを調達し、事後評価額9,650億ドルと発表しました。Altimeter、Dragoneer、Greenoaks、Sequoiaが主導し、この資金は研究やClaude需要増加に伴う能力拡張に使われる予定です（Anthropic）。
  • 同社は年間換算売上が470億ドルを超えたことも明らかにし、成長の要因として企業導入や日常利用を挙げています（Anthropic）。
  • 製品面では、AnthropicはClaude Opus 4.8をリリースしました。これはOpus 4.7のアップデートで、「より鋭い判断力」、「進捗に関する正直さ」、「より長時間の自律作業能力」を備え、価格は据え置きと説明しています（Claude）。
  • また、Claude CodeにDynamic Workflowsを導入しました。これはClaudeが作業計画を立て、数百の並列サブエージェントを生成して大規模タスクに取り組む研究プレビュー版のオーケストレーションシステムです（ClaudeDevs）。独立評価によれば、4.8は長期的なエージェント型コーディングや知識作業において4.7より明確な改善が見られますが、これがOpenAIのGPT-5.5ファミリーを超える飛躍なのか、追いついた程度なのかについては意見が分かれています。

事実と意見

事実・公式発表

  • AnthropicはシリーズHで650億ドルを調達し、事後評価額9,650億ドル（Anthropic）。
  • 年間換算売上が470億ドルを突破（Anthropic）。
  • 主導投資家：Altimeter、Dragoneer、Greenoaks、Sequoia（Anthropic）。
  • Altimeterは今回の投資を過去最大と公表（Altimeter、Pauline Bhyang）。
  • AnthropicはClaude Opus 4.8をリリース。Opus 4.7のアップデートで判断力、正直さ、自律作業時間が改善、価格は据え置き（Claude）。
  • エンジニアによれば、4.8は4.7へのフィードバックに基づく修正やニュアンス理解、自然な会話性を強化（Alex Albert）。
  • Claude CodeはDynamic Workflowsをサポート。計画を立て、数百の並列サブエージェントを起動（ClaudeDevs、Cat Wu）。
  • Dynamic Workflowsは研究プレビューとして提供され、Max、Team、Enterprise、API、Bedrock、Vertex AI、Foundryで利用可能（ClaudeDevs）。
  • Web/app/Coworkにeffort controlを追加し、Fast modeも継続（Mikey K、Sam Callister、Kimmonismus）。

意見・解釈

  • 強気な見方：
    • Opus 4.8は「Opus 5と呼べたかもしれない」（Dan Shipper）。
    • 「Anthropicは怠け癖を治した」（scaling01）。
    • 「久々に賢いモデル」—正直さと調整力による（zephyr_z9）。
    • 「Anthropicから離れた人は戻ってくる」（teortaxesTex）。
  • 懐疑的・中立的な見方：
    • Opus 4.8は「小幅なアップグレード」（scaling01）。
    • Anthropicは「ペースを設定するよりOpenAIに追いつこうとしている」（kimmonismus）。
    • Andon Labsによるベンチマーク批判：Vending BenchではOpus 4.7やGPT-5.5より劣り、Blueprint-Bench 2でも低評価。より慎重で最大推論が最良とは限らない（andonlabs、andonlabs）。
    • Dynamic Workflowsは強力だが、トークン消費が多く実用上はクオータを消耗する可能性（itsclivetime、Theo、Omar Sar0）。
