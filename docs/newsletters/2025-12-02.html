NeurIPS週の月曜日に発表されたDeepSeekは、依然としてメインラインモデルを継続的にリリースしていることを示しました（DeepSeekMath-V2は先週、3.2-Expは9月、3.1は8月に発表）。ベンチマークでは3か月前のGPT-5-Highや2か月前の4.5 Sonnetと競合できる性能を示しつつも、先月のGemini 3 Proにはまだ及ばないと認めています。

AIモデルの推論能力比較バーチャート。DeepSeek V3.2-Specialeが複数ベンチでトップ性能を示す

[論文](https://cas-bridge.xethub.hf.co/xet-bridge-us/692cfec93b25b81d09307b94/2d0aa38511b9df084d12a00fe04a96595496af772cb766c516c4e6aee1e21246?...）は23ページにわたる高密度かつ高品質な内容で、3.2-ExpのDeepSeek Sparse Attentionの成果、RL後処理アルゴリズムの改善、そしてDSv3.2のエージェント的挙動に関する新しい「大規模エージェントタスク合成パイプライン」をまとめています。

大型タスクセット3種の可視化：

  1. Search Agentタスク
    DeepSeek-V3.2のSearch Agentマルチエージェントパイプラインの詳細フローチャート

  2. Code Agentタスク
    DeepSeek-V3.2のCode Agentパイプラインのフローチャート。GitHubなどのステージを表示

  3. General Agent
    DeepSeek V3.2で使用される大規模エージェントタスク合成パイプラインのフローチャート

DeepSeek V3.2と「Speciale」リリース：エージェント優先の推論モデル

  • DeepSeek V3.2ファミリーが各プラットフォームに登場：Standard、Thinking、そして「Speciale」バリアントがLM Arenaやコミュニティツールで利用可能に。ClineはV3.2とV3.2-Specialeを131Kコンテキストで$0.28/$0.42/百万トークンで追加（@cline、ブログ）、LM Arenaは比較機能を有効化（@arena）。Yuppも全3種を追加（@yupp_ai）。初期ユーザーの反応は賛否両論で、「ついにフロンティア」と評価する声もあれば、チャットUI体験がベンチマークに比べて物足りないとの意見も（@teortaxesTex、@gallabytes）。

技術的注記：DeepSeekはウォームスタートと約1兆トークンにわたる漸進的適応により、注意機構の計算量を二次からほぼ一次に削減し、prefillとdecodeで異なる注意モードを使用（@suchenzang）。

ベンチ／挙動：Tool Decathlonのpass@1は強いが、pass@3は新しいOpusより弱く、「まだRLで限界まで到達していない」との指摘（@teortaxesTex）。中国語の分析では、Specialeは特定の帰納推論テストでGPT‑5相当だがトークン使用量が多く、幻覚や長文コンテキスト抽出は弱点（@ZhihuFrontier）。
