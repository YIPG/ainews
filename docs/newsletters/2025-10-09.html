ヒューマノイドロボティクス：Figure 03の発表、機能、業界動向

  • Figureが次世代ヒューマノイド「Figure 03」を発表。高品質なデモ映像とシステム設計・製品目標に関する詳細な解説を公開しました。チームは「この映像に遠隔操作は一切ない」と強調し、「Helix、家庭、そして世界規模での展開」を目指すとしています。発表とフォローアップは@Figure_robot、@adcock_brettから。ロボティクス全体の文脈として、SoftBankがABBのロボティクス部門を54億ドルで買収することもThe Rundown経由で報じられています。
  • 議論：初期レビューではデモの細部（例：仕分けの選択）に関する指摘もありましたが、全体としては能力の進化や「非遠隔操作」という主張が専門家の強い関心を集めました。反応は@Teknium1などから。

オープンフロンティアモデルとリリース：Reflectionの20億ドル調達、Diffusion LM、GLM-4.6、小型モデル推論

  • Reflectionが20億ドルを調達し、オープンウェイトのフロンティアモデル構築へ。大規模MoE事前学習とRLをゼロからスケーリングし、安全性と評価を重視したオープンインテリジェンスのロードマップを掲げています。創業者とチームはAlphaGo、PaLM、Geminiの経験者で、SF/NY/ロンドンで採用中。声明は@reflection_ai、解説は@achowdhery、@ClementDelangueから。
  • Diffusion Language Modelsの大型化（オープン）：Radical NumericsがRND1を公開。30BパラメータのスパースMoE DLM（3Bアクティブ）で、重み・コード・学習詳細を公開し、DLM推論や後処理研究を促進。ARからDiffusionへの変換パイプラインも提供。発表は@RadicalNumerics、要約スレッドは@iScienceLuvrから。
  • ZhipuのGLM-4.6とオープンモデルの勢い：GLM-4.6がDesign Arenaベンチマークで好成績（@Zai_org）。Clineによると、GLM-4.5-AirとQwen3-Coderが同社のエージェントIDEで最も人気のローカルモデル（tweet）。
  • エッジでの小型推論：AI21のJamba Reasoning 3BがIFBenchで52%を記録し「小型」推論モデルのトップに（@AI21Labs）。関連してAlibabaのQwenはQwen3-Omni（ネイティブなエンドツーエンドマルチモーダル）やQwen-Image-Edit 2509を発表し、オープンウェイトモデルで首位に（@Alibaba_Qwen、tweet）。

開発者ツールとエージェントスタック：Claude Codeプラグイン、VS Code AI、Geminiエコシステム

  • Claude Codeがプラグイン対応：AnthropicがClaude Code用のプラグインシステムとマーケットプレイスを提供開始。CLIを更新し「/plugin marketplace add anthropics/claude-code」で追加可能。初期のコミュニティマーケットも登場。詳細は@The_Whole_Daisy、@_catwu。
  • VS Code v1.105（9月リリース）：GitHub MCPレジストリ統合、AIによるマージコンフリクト解決、OS通知、GPT-5-Codexによる思考過程表示などAI中心のUX改善。詳細とライブ配信は@code。
  • GoogleのGeminiプラットフォーム更新：AI Studioに「モデル検索」機能（@GoogleAIStudio）、Gemini CLIのホストドキュメント（@_philschmid）、そして「Gemini Enterprise」がノーコードでエージェント構築・ワークフロー自動化を可能に（@Google、@JeffDean）。
  • エージェントパイプラインにおけるメモリと評価駆動の最適化：Mem0のようなメモリレイヤーのテスト（@helloiamleonie）、DSPy/GEPAでモデルを切り替えコストを20倍削減（@JacksonAtkinsX）、DSPy TS使用デモ（@ryancarson）。

ベンチマークと評価：ARC-AGI、METR時間評価、FrontierMath、ドメイン別ランキング

  • GPT-5 ProがARC-AGIで新SOTA：ARC Prizeによる検証で、ARC-AGI-1で70.2%（$4.78/タスク）、ARC-AGI-2で18.3%（$7.41/タスク）を達成（@arcprize）。
  • エージェント型SWEタスクの時間評価：METRによるとClaude Sonnet 4.5の50%時間到達は約1時間53分（信頼区間50〜235分）で、Sonnet 4より有意に改善もOpus 4.1には及ばず（@METR_Evals）。
  • 数学・推論評価：EpochによるとGemini 2.5「Deep Think」がFrontierMathで新記録（@EpochAIResearch）。ARC-AGIの結果を巡り進捗速度やトレンドラインについて議論（@scaling01、@teortaxesTex）。
  • ビジョン編集・デザインタスク：Qwen Image Edit 2509がオープンウェイトモデルで首位（@Alibaba_Qwen）。GLM-4.6もDesign Arenaで好成績（@Zai_org）。

システム・性能・インフラ：GPUカーネル、推論ベンチマーク、MLX高速化
