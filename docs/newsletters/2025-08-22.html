AI Twitter Recap

インタラクティブな世界シミュレーターとエンボディドトレーニング（Genie 3 + SIMA）

  • DeepMind の Genie 3 ワールドモデル（マルチモーダル、永続的シミュレーション）： @demishassabis のスレッドによると、Genie 3 はテキスト、写真、動画でプロンプト可能なインタラクティブな世界シミュレーターで、高度な空間記憶（カメラ外でも状態が保持される）やリアルタイムアバター制御などの機能を備えています（例1、例2、例3）。また、Genie 3 の可能性についてのポッドキャストも公開されました（リンク）。
  • 生成された世界内でのエージェント学習：DeepMind の SIMA が Genie 生成環境内で学習する様子が紹介され、ワールド生成からエンボディド学習まで AI 内で完結するループが実現されています（@bonniesjli）。
  • シミュレーター活用事例：開発者たちは、データ生成、評価のブートストラップ、ローンチ前の安全性テスト、軌跡分析などにシミュレーションを活用しています（@ShreyaR）。Snowglobe は共有可能な読み取り専用リンクを追加（リンク）、SDK も「近日公開予定」です（リンク）。

オープンウェイトのビジョン＆メディア：Qwen Image Edit がリード、Qwen-VL が上昇、動画モデルが急伸

  • Qwen-Image-Edit（Apache-2.0） → 高性能かつ低コストの画像編集：Alibaba が公開した新しいオープンウェイト画像編集モデルは、Image Editing Arena で ELO 1098（第2位） を記録し、GPT‑4o に匹敵する性能を低価格で実現しました（@ArtificialAnlys、@Alibaba_Qwen）。コミュニティ事例では、局所的な編集やスタイルの忠実性が確認されています（建築デモ）。また、Qwen3 が Qualcomm NPU 上で車やロボットに搭載されて動作（リンク）。Qwen‑VL‑Max は Vision ランキングでトップ20入り（10位タイ）。
    • ツール面では、AI Toolkit が Qwen‑Image‑Edit のファインチューニング（3-bit ARA）に対応し、テキスト埋め込みをキャッシュして単一 5090 で 1024 解像度 LoRA を学習可能に。24GB ターゲットはほぼ到達もまだ安定せず（@ostrisai）。
  • 動画：Kling 2.1「Every Frame in Control」＋新規参入モデル：Kling 2.1 が 開始＆終了フレーム指定機能を追加し、1.6 比で 235% 改善と発表。中間フレームの精密生成が可能に（@Kling_ai、Lovart 連携）。Luma の Ray 2 と Runway Gen‑4 Turbo が Video Arena に登場（詳細）。Google は Gemini アプリで Veo 3 の無料生成3回を提供（Gemini、Google）、さらに Google Photos が自然言語による編集（「車を消す」「良くして」など）に対応（リンク）。

DeepSeek V3.1 ロールアウト：エージェント、大規模ローカル推論、初期 UX

  • リリースと注力分野：DeepSeek v3.1 が複数プラットフォームで利用可能に（Baseten、Yupp）。注力分野は SWE エージェント と 検索エージェント で、将来的には完全な DeepResearch システムを目指す（@teortaxesTex、続報）。
  • Apple Silicon 上でのローカル／クラスタ提供：
    • 単一ノード：4-bit 量子化 v3.1 が M3 Ultra（512GB RAM）で約 21 tok/s、メモリ使用量約 380GB（@ivanfioravanti）。
    • 複数ノード：EXO が TB5 経由 MLX Distributed で Mac Studio 間の線形スケーリングを実証。例：M3 Ultra×2 → 1モデルで14 tok/s、×4 → 2モデルで28 tok/s。EXO 1.0 はオープンソース予定（@MattBeton）。
  • エージェント的コーディング姿勢：複数の報告で、デフォルトは非推論型コーダーが望ましいとされ、推論はエージェントループでコンテキストを消費しやすいとの指摘（@nrehiew_、@Teknium1）。Cline の初期テストでは v3.1 が計画時に「仮定」を置く傾向があり、編集失敗率を追跡中（@cline）。
  • ベンチマークのヒント：Extended NYT Connections で v3.1 の推論は R1 より改善、非推論モードは v3‑0324 を上回る結果（@LechMazur）。
