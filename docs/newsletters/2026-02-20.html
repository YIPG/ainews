トップニュース: Gemini 3.1リリースの事実と反応・意見

Googleは開発者向けプレビューとして位置づけられるGemini 3.1 Proをリリースし、Geminiアプリ、NotebookLM、Gemini API / AI Studio、Vertex AIに展開しました。これはGemini 3 Deep Thinkの「コアインテリジェンス」を実用製品向けにスケールダウンしたものと説明されています。発表では推論能力の大幅な向上（特にARC-AGI-2 = 77.1%）、強力なコーディングやエージェント的ツール利用のベンチマーク（例：SWE-Bench Verified = 80.6%）、幻覚の改善が強調されました。独立したリーダーボードや評価者も高性能とコスト・知能のバランスを確認しましたが、反応スレッドでは(a) 実用的な改善（SVG/ウェブ/UI/コード品質、エージェント的ユースケース）への期待、(b) ベンチマーク狙いと「評価ツイート」への懐疑、(c) GDPval（実世界のエージェントタスク）が他のSOTAスコアに比べて先行していない懸念、(d) ロールアウト時の摩擦（Gemini CLI / Code Assist / Antigravityが利用不可または更新が不一致）などが挙げられました。
