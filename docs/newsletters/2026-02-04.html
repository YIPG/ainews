SOTA Audioモデルがすべてを変える

AI企業がデカコーン（評価額100億ドル以上）に到達した際には、その成長を振り返り称えることを方針としていますが、最近ではその希少性が薄れてきているようです。本日は、Sequoia、a16z、ICONIQがEleven@11ラウンドを主導したニュースがありましたが、すぐにCerebrasの発表にかき消されました。CerebrasはOpenAIとの750MW契約（3年間で100億ドル規模）の後、わずか5か月前に評価額80億ドルだったにもかかわらず、Tiger Globalから評価額230億ドルで10億ドル調達という「ダブル・デカコーン」ラウンドを実現しました。

また、Vibe Codingの1周年を迎え、Andrej氏はAgentic Engineeringを今年の新しいメタとして提案しています。一方で、METRはGPT 5.2 Highを新しい6.6時間の人間タスクモデルとして認定し、Opus 4.5を上回りました。さらに、sama氏はCodexの月間アクティブユーザー数が100万人に到達したことを発表しました。

ビッグテックの製品化：Gemini 3があらゆる場所に（Chrome、アプリ規模、「ゲーム」評価）

  • ChromeサイドパネルにGemini 3搭載：GoogleはGemini 3を搭載した新しいChromeサイドパネル体験を提供開始しました。さらに「Nano Banana」統合（Googleの表現）やその他UX変更も加え、ブラウザワークフローとLLM機能の密接な連携を継続しています（Google）。
  • Geminiのスケールとコスト曲線：Google幹部やアナリストはGeminiの急速な普及と提供コストの大幅削減を強調。Sundar氏は「Gemini 3の採用は他のどのモデルよりも速い」と報告し、Alphabetは年間売上4000億ドル超を達成（@sundarpichai）。別の報告では2025年にGemini提供コストが78%削減されたとされています（financialjuice）。さらに、Geminiアプリは2025年第4四半期に月間アクティブユーザー7億5000万人超を記録（OfficialLoganK）。
  • ゲームによるベンチマーク：Googleはモデルの「ソフトスキル」評価として、Poker、Werewolf、Chessなどのゲームで競わせるKaggle Game Arenaを推進。これは不確実性下での計画・コミュニケーション・意思決定能力を配備前に試験する枠組みです（Googleほか）。

IDE内でのコーディングエージェント統合：VS Code「Agent Sessions」、GitHub Copilot Agents、Codex + Claude

  • VS Codeのエージェント戦略転換：VS Codeは「コーディングエージェントの拠点」と位置づける大型アップデートを実施。ローカル・バックグラウンド・クラウドエージェントを統合するAgent Sessionsワークスペース、Claude + Codex対応、並列サブエージェント、統合ブラウザなどを搭載（VS Code）。
  • GitHub Copilotのモデル/エージェント選択機能：GitHubはCopilot Pro+/EnterpriseでClaudeやOpenAI Codexエージェントを選択可能にし、意図に応じてエージェントを選び既存ワークフロー内で非同期処理を実行できるようにしました（GitHub）。
  • Codexの配布とハーネス詳細：OpenAIはCodexの採用状況（初期50万DL、後に100万アクティブユーザー超）やApp/CLI/Web/IDE統合を発表。共通の「Codexハーネス」をJSON-RPC「Codex App Server」プロトコル経由で提供（OpenAI）。
  • 課題も残る：一部ユーザーはCodexがCPUのみの環境で動作しGPUが使えないと報告し、GPU対応を要望（Yuchenj_UW）。
  • OpenClaw/エージェントコミュニティのプラットフォーム化：OpenClawミートアップやClawHubなどのツール群が、ワークフロー・セキュリティ・配布を中心に急速にプロフェッショナル化。
