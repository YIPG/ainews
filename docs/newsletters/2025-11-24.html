    • Tool Search Tool（遅延ツール読み込み）によりツールコンテキストの冗長性を最大85%削減し、内部評価でツール精度を向上（tweet、tweet）。

    • Tool Use Examples（ツールスキーマに例を埋め込み）で複雑なパラメータ処理精度を72%→90%に改善（tweet、tweet）。
