Microsoft Build: MAI-Thinking-1、モデルファミリー拡張、そしてエージェントネイティブなWindows/GitHub環境

  • Microsoftの新しい社内モデル開発が本日の最大の技術的ニュースです : @mustafasuleyman 氏は MAI-Thinking-1 を紹介しました。これは 35BアクティブパラメータのMoE で、256Kコンテキスト を持ち、AIME 2025で97%、SWE-Bench Proで53%、さらにブラインド比較で Sonnet 4.6 より人間の好みが高いとされています。モデルは MAIA 200 に最適化され、GB200 と比較して性能/コスト、性能/ワットで向上が報告されています。Microsoftは同時に、推論、コード、画像、音声、音声認識をカバーする 7モデルのMAIファミリー を @MicrosoftAI 経由で発表し、OpenRouter、fal、Baseten などのサードパーティからもすぐに利用可能になりました。
  • 詳細な技術レポートが研究者から注目を集めました : 109ページのレポート はスケーリングラダー、MFU、RL/ポストトレーニング、データキュレーションに関する開示レベルの高さが評価され、@eliebakouch、@HannaHajishirzi、@asadovsky らがスレッドで紹介しました。Microsoftが 第三者による蒸留なし を主張している点や、パイプラインの一部で GEPA / DSPyスタイルのプロンプト最適化 を使用している点も、@bj2rn、@lateinteraction、@LakshyAAAgrawal によって指摘されました。
  • Buildではローカル/エージェントデバイスの方向性も強調されました : Project Solara / Scout、Surface RTX Spark Dev Box、ローカルでの1兆パラメータデモなどに関するツイートは、Microsoftがクラウド専用ではなくエージェントネイティブデバイスとローカル推論に注力していることを示しています。これは@TheTuringPost、@kimmonismus、@yusuf_i_mehdi によってまとめられました。ソフトウェア面では、GitHubがCopilotデスクトップアプリを発表し、キャンバス、ローカル/クラウド/モバイル間の継続性、そして「エージェントネイティブなソフトウェア開発」というコンセプトを@pierceboggan、@lukehoban が紹介しました。Microsoftはまた、MAI-Code-1-Flash をVS Code/Copilotに導入しました（@pierceboggan）。

ローカルファーストなコンピュータ利用エージェントとデスクトップエージェントシェル

  • コンピュータ利用エージェントが急速にユーザーデバイスへ移行しています : @hcompany_ai はローカルファーストのコンピュータ利用モデル Holo 3.1 を発表しました。@TeksEdge はこれをQwenベースのGUI/制御モデルと説明し、0.8Bから35Bまでのチェックポイント、NVFP4/FP8/Q4 GGUFなどの最適化フォーマット、ノートPCや小型ワークステーション対応、35B版でAndroidWorldにおいて79.3%のスコアを達成したとまとめています。@NielsRogge もAndroidWorldでのSOTA達成を指摘しました。
  • デスクトップ向けエージェントシェルが増加しています : @NousResearch と@Teknium が Hermes Desktop をリリースし、@cognition はローカルとクラウドエージェントを統合する Devin Desktop を発表しました。@windsurf はこれをACP互換バックエンドでローカル/クラウドエージェントの「艦隊」を管理するためのUIと説明しています。@russelljkaplan はエージェント中立のアプローチを強調し、Devin、Claude Code、Codexとの互換性を明示しました。
  • ローカル/クラウドのハイブリッド実行が標準アーキテクチャになりつつあります : @perplexity_ai はPerplexity Computer向けにハイブリッドエージェント推論を発表し、作業をローカルとクラウドのフロンティアモデルに分割してプライバシー保護とトークン効率向上を図っています。@AravSrinivas はこれがWindowsノートPCに導入され、後にMac/Linuxにも対応すると述べました。Microsoft Buildのメッセージと合わせると、エージェントUXはデスクトップネイティブ化し、推論は選択的にローカル化する傾向が明確です。
