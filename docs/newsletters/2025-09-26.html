Googleの9月アップデート：Gemini Robotics 1.5、Live、Veo 3、Flash価格改定

  • Gemini Robotics 1.5 + Live + Veo 3 GA : Googleは9月に多数のアップデートを発表しました。Gemini Robotics 1.5（高度な推論機能「ER 1.5」を含む）、最新のGemini Live、EmbeddingGemma、Veo 3 GAとAPI更新、AI Edgeギャラリー、Batch API埋め込み対応、Flash/Flash Lite更新、Chrome DevTools MCP、VaultGemmaなどです（@osanseviero）。Robotics-ER 1.5は空間・時間推論に強く、回答精度向上のための「思考」機能を備えています（@_anniexie）。Veo 3はすでにGoogleの音楽ビデオ事例など、制作ワークフローで活用されています（@FlowbyGoogle）。Googleはまた、Labsでレストラン予約エージェントなどのエージェント機能を一般ユーザーに展開しています（@rmstein）。Gemini 2.5 Flashは品質がわずかに向上し、価格は約30％低下しました（@scaling01）。

コードインテリジェンスとエージェント型コーディング

  • MetaのCode World Model (CWM) : 実行トレースやエージェント的なインタラクション（バグ修正、編集、Docker実行）を通じてコードの意味を学習する、オープンウェイトの32Bモデル。Pythonの逐次シミュレーション、マルチターンのソフトウェアタスク、131kコンテキストに対応。SWE-bench Verifiedで65.7%、LiveCodeBenchで68.4%、数学分野でも高成績（Math-500で96.5%、AIME-24で75.8%）。詳細は概要、論文参照。関連アイデアとして、ソースコードとインタプリタ状態を交互に学習させることで意味理解を強制する手法があります（@giffmana）。
  • ローカル優先のコーディング環境 : Qwen3-Coder-30B（AWQ 4-bit）は単一3090で約115トークン/秒を達成（@QuixiAI）。開発者はQwen3-CoderをClineやLM Studioと組み合わせて高品質なローカルコーディングを行っています（@cline、ガイド、ブログ）。Clineは「ワークフロー構築用ワークフロー」機能を追加し（プロンプトレシピ、ブログ）、無料アルファ版でコンテキストを200kから1Mトークンに拡大しました（@cline）。
  • ランタイム/バックエンド : vLLM v1はハイブリッドモデル（Mamba/Mamba2、線形アテンションなど）を第一級として扱い、v0より性能向上（@RedHat_AI）。Apple Silicon向けmlx-lmはハイブリッドSSMやスライディングウィンドウアテンションのバッチ推論を追加し、MetaのCWMもサポートしました（@awnihannun）。

システムとインフラ：カーネル、検索、ホスティング

  • FlashAttention 4解析 : ModalがFA4をリバースエンジニアリングし、約20％の速度向上の理由を解説。特殊なワープレイアウト、softmax用expの三次近似、より積極的な非同期化など。詳細は@charles_irl、ブログ、エンジニアのコメント@bernhardsson、@akshat_b。
