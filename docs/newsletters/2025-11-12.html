漸進的な一歩

GPT 5.1 が本日 ChatGPT にローンチされ、API は「今週後半」に提供予定です。

  • 5.1 Instant は以下の特徴があります：
  • 「デフォルトでより温かみがあり、会話的…遊び心で人を驚かせつつ、明確で有用」
  • 指示の理解力が向上 — emdash の好みも尊重
  • 適応的推論を用いて、難しい質問では回答前に考える時間を取るかどうかを判断し、より徹底的かつ正確な回答を提供しつつ、簡単な質問には迅速に応答
  • 5.1 Thinking は以下の改善があります：
  • 質問に応じて思考時間をより精密に調整

GPT5.0 は「レガシーモデル」となり、3か月後に廃止予定です。

今回のブログ記事には AIME や Codeforces への言及はありますが、評価結果は掲載されておらず、一部から批判が出ています。

ChatGPT にはパーソナライズ用の新しいトーン切替機能も追加されました。Fidji Simo のブログによれば、「8億人以上が ChatGPT を利用しており、もはや一律対応の時代は過ぎた」とのことです。

自律性とフィジカルAI：Waymo高速道路展開、AnthropicのProject Fetch、Perceptronのプラットフォーム

  • Waymo 高速道路運転が一般公開：Waymo は Phoenix、LA、SFベイエリアで高速道路運転を一般利用者向けに展開し、SF↔San Jose 間を SJC 空港まで接続します。経営陣はこれを Driver の汎化能力と安全性の証明と位置づけ、規模拡大により新たな空港ルートや長距離走行が可能になるとしています。発表は @dmitri_dolgov、@JeffDean から。
  • Anthropic の Project Fetch（Claude有無のロボット犬）：Anthropic はロボット工学者でない2チームに四足歩行ロボットのプログラムを依頼し、1チームのみ Claude を使用可能としました。これは「LLMをロボティクスのコパイロットとして活用する」ことの計画・制御作成、デバッグ、反復速度に関する実証的検証です。結果と方法論は @AnthropicAI のスレッドに。
  • Perceptron の「Physical AI」プラットフォーム：マルチモーダルな認識・行動アプリ向けの新しい API と Python SDK を提供。現在 Isaac-0.1 と Qwen3VL‑235B をサポートし、視覚＋言語に基づくプロンプトや「チャット競技」を実装可能。今週は Isaac が無料で利用可能。詳細は @perceptroninc、@AkshatS07。
